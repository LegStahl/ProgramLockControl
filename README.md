# ProgramLockControl
## Description
This device is supposed to use for controlling the door. It can open the door if the right IButton is pressed, also it has a button to open the door.
Device controls magnetic lock. Internally device can coolect info about opening the door, for example: key number, time of enter and etc.
Also device can another control high current till 5A load. This connection is supposed to be used for LED strip.
